#2.3
1. const 와 let
   - const 는 constant(상수)라는 것으로 값이 바뀔 수 없음을 의미

변수만들때 let, const, var차이
let 재선언 금지, 재할당 가능
const 재선언 금지, 재할당 금지
var 재선언 가능, 재할당 가능

let a = b;
let a = c;
//재선언 금지

let a = b;
a = c;
//재할당은 가능

const a = b;
const a = c;
//재선언 금지

const a = b;
a = c;
//재할당 금지

var a = b;
var a = c;
a = d;
//재선언, 재할당 가능

*변수 선언 규칙
- 기본적으로 const를 쓰고 필요할때만 let을 쓰되, var는 쓰지 말 것!
